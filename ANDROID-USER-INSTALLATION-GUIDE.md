## â“ Troubleshooting

### Problem: Bells don't ring when app is closed

**Solution:**
1. Check battery optimization is disabled (Step 5)
2. Make sure all permissions are granted
3. On Xiaomi/Huawei: Enable autostart

### Problem: Bells ring late

**Solution:**
1. Disable battery saver mode
2. Keep app unrestricted in battery settings
3. Don't use "Power saving mode"

### Problem: No sound when bell rings

**Solution:**
1. Check phone volume (notification volume)
2. Check Do Not Disturb is off
3. Test bell sound in app settings

### Problem: App stops after restarting phone

**Solution:**
1. Enable autostart (Xiaomi/Huawei phones)
2. Check battery optimization is disabled
3. Reinstall app if needed

### Problem: Can't install APK

**Solution:**
1. Make sure "Unknown sources" is enabled
2. Try installing from different file manager
3. Check you have enough storage space
