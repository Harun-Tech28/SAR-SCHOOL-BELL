# SchoolBell Automation System - Project Description

## üéì Project Overview

**SchoolBell** is a modern, web-based school automation platform that revolutionizes how educational institutions manage daily operations. It combines traditional bell scheduling with cutting-edge AI voice technology to deliver professional announcements, automated prayer calls, and comprehensive student management.

## üéØ Problem Statement

Schools face several challenges in managing daily operations:
- Manual bell ringing is time-consuming and inconsistent
- Announcements lack professionalism and clarity
- Prayer time management requires constant attention
- Student calling systems are inefficient
- Traditional systems aren't mobile-friendly
- Multilingual support is limited

## üí° Solution

SchoolBell provides an all-in-one automation platform that:
- **Automates bell schedules** with 13 different bell types
- **Generates professional AI voice announcements** in multiple languages
- **Manages Islamic prayer times** with authentic Adhan calls
- **Streamlines student management** and calling systems
- **Works seamlessly on mobile and desktop** devices
- **Supports multiple languages** (English, Hausa, Twi, Arabic)

## üåü Core Features

### 1. Intelligent Timetable System
- Create unlimited bell schedules
- Choose from 13 bell types (traditional, electronic, chimes, Islamic)
- Auto-generate or customize voice announcements
- Select from multiple AI voices
- Daily or specific day scheduling

### 2. AI-Powered Voice System
- **3 Voice Providers**: OpenAI, Azure, ElevenLabs
- **8 Voice Options**: Professional, authoritative, friendly, Islamic voices
- **4 Languages**: English, Hausa, Twi, Arabic
- High-quality audio synthesis
- Smart voice recommendations

### 3. Islamic Prayer Integration
- Automated Adhan (call to prayer)
- Five daily prayer times
- Authentic Islamic voice
- Customizable prayer schedules
- Real-time prayer time tracking

### 4. Student Management
- Comprehensive student database
- Class organization (JHS 1-3, custom classes)
- Quick search functionality
- Individual student calling with AI voice

### 5. Mobile-Responsive Design
- Full-screen mobile menu with grid layout
- Touch-optimized interface
- Responsive dashboard
- Works on all devices
