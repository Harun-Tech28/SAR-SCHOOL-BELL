================================================================================
                        QUICK START
================================================================================

FIRST TIME SETUP (Do this once):
---------------------------------
1. Install Node.js from: https://nodejs.org (download LTS version)
2. Restart your computer
3. Double-click: SETUP-FOR-CLIENT.bat
4. Wait 5-10 minutes for setup to complete
5. App will start automatically when ready

DAILY USE (After setup):
------------------------
1. Double-click: Start Ghana School Bell.bat
2. The app will open and run in the background
3. Bells will play automatically according to your schedule

TO STOP THE APP:
----------------
1. Look for the bell icon in the system tray (bottom-right corner)
2. Right-click the icon
3. Click "Quit App"

================================================================================
                        TROUBLESHOOTING
================================================================================

Problem: "npm is not recognized" error
Solution: Install Node.js and restart computer

Problem: App won't start
Solution: Run SETUP-FOR-CLIENT.bat again

Problem: No sound
Solution: Check volume, test speakers, check Windows sound settings

Problem: Timetable not saving
Solution: Click "Save" after changes, don't force-close the app

For more help, see the TROUBLESHOOTING section in CLIENT_USER_GUIDE.md
